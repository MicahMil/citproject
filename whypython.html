Why would you choose Python?

It's Simple.  Python is simple programming language. Its syntax feels more natual than other programming programming languages. Becuase it's a high level programming langugae, it reads like English which takes away some of the stress of learning the syntax. It takes away some of the worry about the details of programming and lets them focus more on the concepts of programming.

It's Not Strict.  Python is not as strict as other programming languages. For example, it's not nesecerry to define the type of variable in Python, and its also not necessary to add a semicolon at the end of a statement. Python makes you follow good programming practice such as proper indentation. Small things like this can make learning easier for beginners.

It's Expressive.  Python enables you to write programs with few lines of code.

Why would you not choose Python?

Lack of Mobile Support.  There are some reasons why one might not choose Python for a project. One reason is the lack of mobile support that Python offers. It has been stagnate while other languages have excelled in this area.

Slower Programs.  Since Python is an interpreted language, the code has to be converted into machine language at runtime which makes for a slower program.

Bad Habits.   Programmers who start by learning Python don't get some of the basic principles of other programming languages such as using brackets and semicolons.
